# Visualizing Coffee Quality and Sensory Characteristics

This project explores the quality and sensory characteristics of coffee using advanced data visualization in R, with a focus on understanding what drives high coffee ratings, how coffee quality differs across countries, and how processing methods relate to quality.

## Project Overview

The analysis is based on a cleaned coffee quality dataset containing information on over 1,300 coffee samples, including sensory attributes, origin, processing methods, and total cup scores. The project combines static and interactive visualizations to examine coffee quality from multiple perspectives.
